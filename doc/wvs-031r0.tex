To compute relative humidity in the full forward model, the following
transformations of $\beta$ and $\frac{\d \beta}{\d T}$ are applied:

\begin{equation}\begin{split}
e_s =\,& \exp\left(19.6783-\frac{6140.4}T\right)\\
\beta^{\text{RHi}} =\,& \beta^{\text{H}_2\text{O}} \frac{e_s}P\\
\frac{\d \beta^{\text{RHi}}}{\d T} =\,&
 \frac{e_s}P \frac{\d \beta^{\text{H}_2\text{O}}}{\d T} +
 \frac{6140.4}{T^2} \beta^{\text{RHi}}
\end{split}\end{equation}

To request this transformation in the forward model configuration, define a
molecule group with a group name {\tt RHI} and members the desired water
molecules.  {\tt RHI} is defined to be a molecule, but there is no spectroscopy
information for it, so (as a molecule) it can only be used as a molecule group
name.

It is possible to retrieve relative humidity and specific humidity in the same
run.  The forward model configurator will snivel about duplicate molecules, but
will nevertheless continue.
