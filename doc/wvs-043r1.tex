The relation between $\phi$ and $H$ is

\begin{equation}\label{one}
\frac{\text{d} \phi}{\text{d} H} = \frac1H
 \frac{\mathcal{N}_0 H_0}{\sqrt{\mathcal{N}^2 H^2 - \mathcal{N}_0^2 H_0^2}}
\end{equation}

where $\mathcal{N}$ is a given (tabulated) function of $H$ and the zero
subscript applies at the tangent point.  From this, given $H$ at the same
(tabulated) points as $\mathcal{N}$,

\begin{equation}
\phi_{i+1} = \phi_i + \int_{H_i}^{H_{i+1}} \text{d}H \, \frac1H
\frac{\mathcal{N}_0 H_0}{\sqrt{\mathcal{N}^2 H^2 - \mathcal{N}_0^2 H_0^2}} \,.
\end{equation}

As $\mathcal{N} \rightarrow 1$, using $H = H_0 \sec \phi$, this goes to
$\phi_{i+1} = \phi_i + \int_{\phi_i}^{\phi_{i+1}} \text{d} \phi$. If we are
instead given $\phi$ and want $H$ we replace $H = H_0 \sec \phi$ and write

\begin{equation}
\frac{\text{d} H}{\text{d}\phi} = \frac{H_0}{N_0}
 \frac{\sqrt{\mathcal{N}^2 - \mathcal{N}_0^2 \cos^2 \phi}}
      {\cos^2 \phi}
\end{equation}

where $\mathcal{N}$ is now considered to be a given function of $\phi$.  From
this, given $\mathcal{N}$ and $\phi$ we have

\begin{equation}
H_{i+1} = H_i + \int_{\phi_i}^{\phi_{i+1}} \text{d} \phi \, \frac{H_0}{N_0}
 \frac{\sqrt{\mathcal{N}^2 - \mathcal{N}_0^2 \cos^2 \phi}}
      {\cos^2 \phi} \,.
\end{equation}

Again, using $H = H_0 \sec \phi$, as $\mathcal{N} \rightarrow 1$, this goes to
$H_{i+1} = H_i + \int_{H_i}^{H_{i+1}} \text{d} H$.

Instead of Equation (\ref{one}), we compute the correction to the geometric
$\phi$ due to refraction,

\begin{equation}
\frac{\text{d} \Delta \phi}{\text{d} H} = \frac1H \left[
 \frac{\mathcal{N}_0 H_0}{\sqrt{\mathcal{N}^2 H^2 - \mathcal{N}_0^2 H_0^2}} -
 \frac{H_0}{\sqrt{H^2 - H_0^2}} \right]\,,
\end{equation}

from which

\begin{equation}
\Delta \phi_{i+1} = \int_{H_i}^{H_{i+1}} \text{d}H \, \frac1H \left[
 \frac{\mathcal{N}_0 H_0}{\sqrt{\mathcal{N}^2 H^2 - \mathcal{N}_0^2 H_0^2}} -
 \frac{H_0}{\sqrt{H^2 - H_0^2}} \right] \,,
\end{equation}

which goes to zero as $\mathcal{N} \rightarrow 1$. Similarly, we want to compute

\begin{equation}
\Delta H_{i+1} = \int_{\phi_i}^{\phi_{i+1}} \text{d} \phi\, H_0 \left[
 \frac{\sqrt{\mathcal{N}^2 - \mathcal{N}_0^2 \cos^2 \phi}}
      {\mathcal{N}_0 \cos^2 \phi} -
 \frac{\sin \phi}{\cos^2 \phi} \right]
  \,,
\end{equation}

which also goes to zero as $\mathcal{N} \rightarrow 1$.
